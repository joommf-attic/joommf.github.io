joommf Interactive Micromagnetic simulation in Jupyter
The Joommf project aims to integrate the popular Micromagnetic simulation package OOMMF with the Jupyter Notebook and Python. It is currently being developed by Hans Fangohr, Marijan Beg and Ryan Pepper at the University of Southampton.

Easy to Script
The use of Python allows non-specialist programmers to quickly get up to speed and run simulations.

Powerful
OOMMF is parallelised using OpenMP, allowing users to take advantage of multiple processors on shared-memory machines.

Cross Platform
Runs on Linux, OS X and Windows.

The project is broken into several independent toolkits:

  • oommfc - a low-level Python wrapper for running OOMMF simulations.
  • oommffield - a Python module for reading, writing and plotting OMF files, outputted by OOMMF and other micromagnetics packages.
  • oommfodt - a Python module which allows users to load ODT files into Pandas DataFrames.
Currently we are in early development, which means that changes may happen rapidly and break backwards compatibility. To get details about the project, or ask questions we have a Google Group.

Joommf is part of the EU Horizon 2020 project OpenDreamKit.
