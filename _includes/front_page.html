The JOOMMF project aims to integrate the popular micromagnetic simulation package OOMMF with the Jupyter Notebook and Python. It is currently being developed by Hans Fangohr, Marijan Beg and Ryan Pepper at the University of Southampton.

Thanks go to Thomas Kluyver, David Cortes, Mark Vousden, and Rebecca Carey for helping with testing and improvements of software and webpages.

JOOMMF is part of the EU Horizon 2020 project OpenDreamKit.

EASY TO SCRIPT  

The use of Python allows non-specialist programmers to quickly get up to speed and run simulations.

Based on powerful OOMMF  

OOMMF is parallelised using OpenMP, allowing users to take advantage of multiple processors on shared-memory machines.

Cross Platform  

Ambition to execute on Linux, OS X and Windows (need Windows testers).

To learn more, check the Get Started section

Currently we are in early development, which means that changes may happen rapidly and break backwards compatibility.
